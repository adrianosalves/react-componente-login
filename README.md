# react-componente-login
Componente de Login

Com prazer! Aqui está um resumo do desafio que você concluiu, dividido em componentes e funcionalidades:
Componentes Criados:

    Header: Exibe o título da aplicação.
    Botão: Componente reutilizável que aceita a função onClick como propriedade.
    Card: Componente que encapsula o formulário de login, centralizando a chamada do formulário em um único componente na página inicial.

Funcionalidade de Boas-Vindas:

    Implementação: Ao clicar no botão, um alerta exibe uma mensagem de boas-vindas.
    Teste Unitário: Criado para garantir o funcionamento correto da função de boas-vindas.

Resumo:
Você estruturou a página inicial em componentes reutilizáveis (Header, Botão, Card) e implementou uma funcionalidade de boas-vindas com teste unitário. O componente Card foi refatorado para incluir o formulário de login, simplificando a organização da página inicial.
